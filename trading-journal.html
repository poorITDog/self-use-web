每月固定支出 月費

例如租金、健身房月費、訂閱：揀檢視月份時若該月尚未有對應一筆，會自動加支出（金額跟範本嘅「分段金額」：幾多錢由邊個月起）。已經寫入流水嘅舊月唔會被程式改數；要改舊月請單筆刪除再揀返該月觸發補帳，或手動改流水。每月幾號建議 1–28。

調整範本「分段金額」

適用於月費／固定收入：填新金額同生效月份（含該月）後，之後首次自動補帳會用新段；舊月已存在嘅流水唔會自動改。例：租三月仍 4000、四月起 5000 → 新金額填 5000、生效月填 2025-04。

每月固定收入 入帳

例如出糧、固定股息：揀月份檢視時若該月未有一筆會自動加收入。金額邏輯同月費（分段、唔改寫舊月流水）。

Google Drive 雲端備份（個人）

用 Google Apps Script 將成份日誌存做 Drive 上一個 JSON 檔；多部機用同一 URL + Token 即可同步（詳見專案內 journal-google-sync/README.md）。

建議用 http://localhost 開本頁再做雲端 fetch；file:// 有機會被瀏覽器擋跨網域請求。

關於「自動合併同步」：舊版曾可喺此處剔選，喺你儲存資料後約 5 秒自動呼叫「雲端合併同步」（拉雲端 → 合併 → 上傳）。目前功能已暫停，避免以為有背景同步但實際冇做嘢；請繼續用下面手動「雲端合併同步」。

合併同步（預設）：會先下載雲端，將交易、生活目標／日誌、錢包／時間表、工作日曆、每日心情日記、自訂每日金句／效應池、效應庫（閃卡／出現紀錄）、交易思路庫、交易筆記閃卡（主題／卡片）、待辦快速備忘、XAU 模擬倉等一併按規則合併（交易／帶 createdAt 嘅紀錄：同 ID 保留較新；快速備忘同思路庫：同 ID 以較新 updatedAt 為準；設定（含 Playbook 標籤庫、策略說明、犯錯標籤、範本）：標籤名單會合併去重；同一 Playbook 標籤嘅策略說明以較新 updatedAt 為準；錢包預設／時間格：邊邊 syncUpdatedAt 新就邊邊主導預算欄；XAU 開倉狀態以較新 openedAt 為準）。XAU 本地蠟燭時間框／TradingView 時間框只留本機、唔寫入備份 JSON。再寫回本機並上傳成個備份。「從雲端覆蓋本機」才會唔合併整份取代本機（舊備份冇嘅欄位會保留本機）。進階 DELETE 仍係刻意清空雲端專用。
