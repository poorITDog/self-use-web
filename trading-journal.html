Google Drive 雲端備份（個人）

用 Google Apps Script 將成份日誌存做 Drive 上一個 JSON 檔；多部機用同一 URL + Token 即可同步（詳見專案內 journal-google-sync/README.md）。

建議用 http://localhost 開本頁再做雲端 fetch；file:// 有機會被瀏覽器擋跨網域請求。

合併同步（預設）：會先下載雲端，將交易、生活目標／日誌、錢包／時間表、XAU 模擬倉等一併按規則合併（交易／帶 createdAt 嘅紀錄：同 ID 保留較新；錢包預設／時間格：邊邊 syncUpdatedAt 新就邊邊主導預算欄；XAU 開倉狀態以較新 openedAt 為準）。XAU 圖表時間框只留本機、唔上雲。再寫回本機並上傳成個備份 JSON。「從雲端覆蓋本機」才會唔合併整份取代本機（舊備份冇嘅欄位會保留本機）。進階 DELETE 仍係刻意清空雲端專用。
